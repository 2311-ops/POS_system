# FashionPOS

FashionPOS is an offline-first WPF point-of-sale and inventory system for fashion and clothing businesses.

## What it does

- Processes sales at the POS
- Tracks inventory, stock, categories, and collections
- Supports cashier, manager, and owner roles
- Handles event or bazar sales sessions
- Imports products from Excel, CSV, and Shopify exports
- Generates invoices and reports

## Main screens

- Login
- Dashboard
- POS
- Inventory
- Users
- Event mode
- Reports
